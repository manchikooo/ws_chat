# 💬 WS.Chat

Чат для изучения WebSockets. Построен на **React + TypeScript + Vite**.

## 📋 О проекте

Учебный проект, созданный для знакомства с технологией WebSockets в экосистеме React. 

Этот чат позволяет:
- Подключаться к WebSocket серверу
- Отправлять и получать сообщения в реальном времени
- Отображать статус подключения
- Изучать управление WebSocket-соединением в React-компонентах

## 🛠 Стек технологий

| Компонент | Технология |
|-----------|------------|
| Фреймворк | React |
| Язык | TypeScript |
| Сборщик | Vite |
| WebSocket | Native WebSocket API |
